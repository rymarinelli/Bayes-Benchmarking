# Bayes-Benchmarking
This repo is a proof-of-concept project to demonstrate how Bayesian statisticans can be used to inform more effective benchmarking. It uses the Noreval benchmark and compares two verisons of the same model Normistral 7B and Normsistral 11B. 

A Google Colab notebook is shared with the results of the benchmarking of the 7b and 11B.
